114-1 ACV homework 
# How to build?
## Prerequisite
Make sure OpenCV has installed in ur PC
```
opencv 4.5.4(sudo apt install libopencv-dev)
```
### Build and Run
Replace X with the number of HW index
```
cd HW#X_114318047/code/
cmake -S . -B build
cmake --build build -j
./build/HWX
./build/HWX_opencv
```
